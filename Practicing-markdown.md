# MARKDOWN markup language

## Titles exemples:
# Titol h1
## Titol h2
### Titol h3
#### Titol h4
##### Titol h5
###### Titol h6
####### Titol h7


## Bold

This is a **Bold** text. OR This is a __Bold__ text.

## Italic
This is a *Italic* text. OR This is a _Italic_ text.

## Strikethrough
This is a ~~Italic~~ text.
